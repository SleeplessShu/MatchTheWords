**MatchTheWords** - Android-игра на запоминание иностранных слов.  
Игроку нужно находить пары: слово на испанском и его перевод на русский. Приложение помогает учить язык в формате быстрой, увлекательной практики.

---

## 🎮 Возможности

- Поддержка Испанского, Русского, Немецкого и Французского языков
- Оксфордский словарь для уровней А1 и А2
- Группировка слов по темам
- Случайная генерация пар слов
- Подсчёт очков и экран результатов
- Настройки с выбором темы и языка интерфейса

---

## 🛠️ Стек технологий

Kotlin, Python, MVP, SharedPreferences, RecyclerView, Fragment, ViewBinding, SQLite

## 🚀 Как запустить

1. Склонируйте проект:
   ```bash
   git clone https://github.com/SleeplessShu/MatchTheWords.git
   ```
2. Откройте в Android Studio
3. Соберите и запустите на эмуляторе или устройстве

---

## 📂 Структура проекта

- `game/` - логика игры и отображение карточек
- `score/` - экран с результатами
- `settings/` - настройки приложения
- `di/` - внедрение зависимостей
- `utils/` - вспомогательные классы
